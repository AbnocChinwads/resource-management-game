Resource Management Game
========================

A browser-based resource management game built as a personal learning project. Players begin by manually gathering resources before expanding their settlement through buildings, population growth, and worker automation.

Current Status
--------------

The project is in active development and is not yet feature complete.

Current priorities are:

-   Completing account management features.
-   Ensuring the core gameplay systems work as intended.
-   Creating a solid and enjoyable gameplay loop.

Visual polish and interface improvements are intentionally being postponed until the gameplay mechanics are stable.

Current Features
----------------

-   User registration and login
-   Change player name
-   Change password
-   Change email address
-   Email verification
-   Persistent player data
-   Resource gathering
-   Building construction
-   Recipe discovery
-   Worker assignment
-   Welcome emails on registration
-   Automated deployment to a self-hosted server

Planned Features
----------------

### Account Management

-   Delete account and gameplay data

### Gameplay

-   Worker automation
-   Food consumption and population simulation
-   Gameplay balancing

### User Interface

-   Improved layout and navigation
-   Visual polish
-   Responsive design

Project Philosophy
------------------

The focus of this project is to build reliable gameplay systems before investing significant time in presentation. Features are implemented incrementally so they can be tested and understood before additional complexity is introduced.
